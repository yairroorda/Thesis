# Install

- Make sure you have conda installed on you system
- Setup PDAL by running the command below acoording to the [docs](https://pdal.io/en/2.8.4/quickstart.html)
    
    ```conda create --yes --name <myenv> --channel conda-forge pdal```
- Activate the evironment you created and you should be good to go

    ```conda activate <myenv>```
- View the generate point clouds in your favorite point cloud viewer, I recommend [Cloud Compare](https://www.cloudcompare.org/)
